# Mandelbrot Set

This is an implementation of the Mandelbrot Set fractal in MatLab.

## What is the Mandelbrot set?

The Mandelbrot set is generated by iteration, which means to repeat a process over and over again. In mathematics this process is most often the application of a mathematical function.

To know more about the Mandelbrot Set please check this [Article](https://plus.maths.org/content/what-mandelbrot-set).
